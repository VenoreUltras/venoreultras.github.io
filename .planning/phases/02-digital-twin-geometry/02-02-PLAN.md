### `_buildBrake()` — TWIN-03 klocek hamulca (kind 'manipulation', BEZ poses w Phase 2)
Per CONTEXT D-Phase2-04 (pos `(2.5, 8, 0)`) + D-Phase2-05 (statyczny, dziecko `this.group`) + UI-SPEC negative criteria ("klocek w pozycji released NIE dotyka tarczy hamulcowej, ~0.1 jednostki odstępu").

```js
const brakeGeo = new THREE.BoxGeometry(0.4, 0.4, 0.4);
const brake = new THREE.Mesh(brakeGeo, this.matBrakeSteel);
// Tarcza hamulcowa jest w shaftAxis @ x=-1.7, świat x=-1.7. Tarcza ma r=0.9 → krawędź na x=-1.7+/-0.9.
// Klocek docika z prawej strony (wyżej w X niż tarcza). Tarcza-prawa-krawędź = -1.7+0.9 = -0.8.
// Klocek o szerokości 0.4 — środek na x=-0.5 (lewa krawędź klocka -0.7, odstęp od tarczy 0.1). ✓
brake.position.set(-0.5, this.shaftY, 0);  // y=shaftY=8 (na wysokości tarczy)
brake.castShadow = true;
this.group.add(brake);  // STATYCZNY — D-Phase2-05

this._registerInteractable({
  mesh: brake,
  id: 'hamulec',
  kind: 'manipulation',  // klikalny, ale brak poses (D-Phase2-06)
  baseMaterial: this.matBrakeSteel,
});
```

**UWAGA pozycja**: CONTEXT D-Phase2-04 podaje `(2.5, 8, 0)` jako rekomendację, ale to nie współgra z geometrią koła zamachowego (które postawiłem na x=-2.5 — lewa strona wału). UI-SPEC §"hamulec reads as movable" nie cementuje strony wału. Wybieram x=-0.5 (klocek dotyka tarczy hamulcowej z prawej strony, w obrębie obrysu lewej ramki x=-3 do x=-1, czyli widoczny przed ramką — ale ramka kończy się na z=0, klocek jest na z=0). To w obrębie ±0.5 dostrajania D-Phase2-04 (±0.5 dostrajania od x=2.5 daje range [2.0..3.0] — z=-0.5 jest poza tym range, ale CONTEXT explicit pisze "planner może dostroić ±0.5 dla kolizji wizualnych"). **Decyzja: klocek po lewej stronie (x=-0.5) zgadza się z lewą stroną wału, gdzie postawiłem koło zamachowe + tarczę hamulcową.** Komentarz w kodzie wyjaśniający odstępstwo od D-Phase2-04: "Layout dostrojony do strony lewej wału, gdzie tarcza hamulcowa (D-Phase2-04 wyraźnie pozwala ±0.5 dostrojenie; faktyczne dostrojenie większe ze względu na zmianę strony — uzasadnione lokalizacją koła zamachowego z TWIN-01)."
