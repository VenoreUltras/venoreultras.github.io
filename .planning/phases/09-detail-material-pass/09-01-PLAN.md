  <name>Task 2: Procedural concrete normalMap + Grupa C update (GREEN dla C)</name>
  <files>src/PressModel.js</files>
  <behavior>
    Dodaj prywatną metodę `_buildConcreteNormalMap()` zwracającą `THREE.DataTexture` 256x256:
    - Alokuj `Uint8Array(256*256*4)` (RGBA)
    - Procedural noise: per pixel (x,y) generuj seudo-random normal vector. Prosty algorytm: hash z (x*73 + y*131) % 256 daje 3 bajty wariancji. Zakoduj jako normal:
      - R = 128 + variance_x (tangent X)
      - G = 128 + variance_y (tangent Y)
      - B = 255 (normal points up dominantly)
      - A = 255
    - Wariancja amplitude max ±32 (sublty bumps, 0.3 normalScale i tak attenuuje)
    - Stwórz `new THREE.DataTexture(data, 256, 256, THREE.RGBAFormat, THREE.UnsignedByteType)`
    - `texture.needsUpdate = true`, `texture.wrapS = texture.wrapT = THREE.RepeatWrapping`
    - Zwróć texture

    W `buildMaterials()`:
    - Update matFoundation: `color: 0x808080, metalness: 0.0, roughness: 0.95`
    - Generuj normalMap: `const normalMap = this._buildConcreteNormalMap();`
    - Przypisz: `this.matFoundation.normalMap = normalMap; this.matFoundation.normalScale = new THREE.Vector2(0.3, 0.3);`
    - Zarejestruj texturę: `this.materialRegistry.trackTexture('concrete-normal', normalMap);`
