| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-17-02 | Tampering (XSS) | _renderQuestion / _renderScore DOM writes | mitigate | All dynamic values (question, options, explanation, normRef, score) assigned via textContent; static skeleton innerHTML contains no interpolated data. Verified by boundaries.test.js convention + grep. |
