CRITICAL behavioral note: submitAnswer increments currentIndex immediately. So after a user
answers, the store's currentIndex already points at the NEXT question. The per-question feedback
(explanation + normRef) must therefore be rendered by QuizController BEFORE advancing — see Plan 02.
This affects the test scaffold's feedback assertions: feedback is shown on the answered question,
and the "Dalej" button is what re-renders the next question.
